# Set up Entrez
Entrez.email = "researcher@example.com"

# Retrieve article details
print("Retrieving article details from PubMed...")
print("="*80 + "\n")

articles_data = []

for i, pmid in enumerate(pmid_list, 1):
    try:
        print(f"[{i}/{len(pmid_list)}] Fetching PMID: {pmid}")
        
        # Fetch article details
        handle = Entrez.efetch(db="pubmed", id=pmid, rettype="medline", retmode="xml")
        records = Entrez.read(handle)
        handle.close()
        
        if records['PubmedArticle']:
            article = records['PubmedArticle'][0]
            medline = article['MedlineCitation']
            
            # Extract relevant information
            article_info = {
                'PMID': pmid,
                'Title': medline['Article']['ArticleTitle'] if 'ArticleTitle' in medline['Article'] else '',
                'Abstract': '',
                'Journal': medline['Article']['Journal']['Title'] if 'Journal' in medline['Article'] else '',
                'PubDate': ''
            }
            
            # Extract abstract
            if 'Abstract' in medline['Article']:
                abstract_texts = medline['Article']['Abstract']['AbstractText']
                if isinstance(abstract_texts, list):
                    article_info['Abstract'] = ' '.join([str(text) for text in abstract_texts])
                else:
                    article_info['Abstract'] = str(abstract_texts)
            
            # Extract publication date
            if 'DateCompleted' in medline:
                date = medline['DateCompleted']
                article_info['PubDate'] = f"{date.get('Year', '')}-{date.get('Month', '')}-{date.get('Day', '')}"
            
            articles_data.append(article_info)
            print(f"  ✓ Title: {article_info['Title'][:80]}...")
            
        time.sleep(0.5)  # Be respectful to NCBI servers
        
    except Exception as e:
        print(f"  ✗ Error fetching PMID {pmid}: {e}")
        articles_data.append({'PMID': pmid, 'Title': '', 'Abstract': '', 'Journal': '', 'PubDate': ''})

print(f"\n✓ Retrieved details for {len(articles_data)} articles")

# Save articles data
with open(os.path.join(tmp_dir, "articles_data.json"), "w") as f:

Full Abstract:
One of the most common cancers is hepatocellular carcinoma (HCC). Numerous studies have shown the relationship between abnormal lipid metabolism-related genes (LMRGs) and malignancies. This study aims to develop a novel LMRG prognostic model for HCC patients and to study its utility for predictive, preventive, and personalized medicine. By using the least absolute selection and shrinkage operator (LASSO) regression machine learning algorithm, we constructed a risk prognosis model with six LMRGs (<i>AKR1C1</i>, <i>CYP27A1</i>, <i>CYP2C9</i>, <i>GLB1</i>, <i>HMGCS2</i>, and <i>PLPP1</i>). Further investigation of the association between the LMRG model and somatic tumor mutational burden (TMB), tumor immune infiltration, and biological function was performed. We found that the most frequent somatic mutations in the LMRG high-risk group were <i>CTNNB1</i>, <i>TTN</i>, <i>TP53</i>, <i>ALB</i>, <i>MUC16</i>, and <i>PCLO</i>.
