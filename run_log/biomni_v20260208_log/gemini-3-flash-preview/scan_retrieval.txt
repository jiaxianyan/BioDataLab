### Step 4 & 5: Evaluate studies and create final list

Now I will load the extracted metadata and evaluate each article against the criteria:
(i) availability of detailed sample records;
(ii) availability of sequencing library construction methods;
(iii) availability of cell type annotation files or supporting information such as expected cell types or markers.

Since I only have titles and abstracts (and MeSH terms), I will search for keywords that indicate these criteria are met. In a real-world scenario, I might need to fetch the full text, but based on the task constraints and typical retrieval workflows, I will perform a keyword-based assessment of the abstract/metadata which is the standard approach for this kind of "scan retrieval" task unless full-text access tools are specified.

Criteria check logic:
- Sample records: Keywords like "samples", "patients", "individuals", "donors", "tissue", "clinical", "cohort".
- Library construction: Keywords like "10x", "Smart-seq", "library", "sequencing", "Drop-seq", "InDrop", "preparation".
- Cell type annotation: Keywords like "cell type", "cluster", "annotation", "marker", "subpopulation", "atlas", "identified".

Actually, the task implies checking if these are *available* in the study. Usually, papers mentioning scRNA-seq in the abstract will also mention the library type and the cell types found.

import json
import os

tmp_dir = "/root/autodl-tmp/biomni_v20260208_tmp/gemini-3-flash-preview/scan_retrieval/"
metadata_path = os.path.join(tmp_dir, "extracted_metadata.json")

with open(metadata_path, "r") as f:
    articles = json.load(f)

def evaluate_article(article):
    text = (article['Title'] + " " + article['Abstract']).lower()
    
    # (i) Detailed sample records
    # Usually indicated by mentioning the source of samples and quantities
    has_samples = any(kw in text for kw in ["sample", "donor", "patient", "tissue", "collected from", "obtained from"])
    
    # (ii) Library construction methods
    # Mentioning specific technologies or "library construction/preparation"
    has_library = any(kw in text for kw in ["10x", "smart-seq", "sequencing library", "single-cell library", "droplet-based", "chromium"])
    
    # (iii) Cell type annotation or markers
    # Mentioning identifying cell types, markers, or clustering
    has_annotation = any(kw in text for kw in ["cell type", "marker", "cluster", "annotation", "identified", "atlas", "subpopulation"])
    
    # For scRNA-seq papers in PubMed that pass the initial query, they usually meet these if they are actual primary studies.
    # However, we must be strict.
    return has_samples and has_library and has_annotation

results = []
