Identify expression-derived Quantitative Trait Loci (edQTLs) that overlap with Genome-Wide Association Study (GWAS) tag SNPs. 

Using the provided GWAS SNP list in `benchmark/dataset/GPEdit/gwas_snps.tsv` and the edQTL list in `benchmark/dataset/GPEdit/edqtls.tsv`, filter for edQTLs whose genomic range (`start` to `end` on a specific `chr`) contains a GWAS SNP''s position (`pos`). 

Save the complete rows of the identified overlapping edQTLs to a tab-separated file at `/root/autodl-tmp/biomni_v20260208_pred_results/doubao-seed-1-8/gpedit_refinement.tsv`.

Output example format:
```
chr	start	end	edqtl_id	gene
chr1	100000	105000	edqtl_1	GeneA
chr2	200000	205000	edqtl_2	GeneB
``` 

All intermediate results should be saved in `/root/autodl-tmp/biomni_v20260208_tmp/doubao-seed-1-8/gpedit_refinement` directory.

<execute>
import pandas as pd
import os

# Create intermediate directory if it doesn't exist
intermediate_dir = "/root/autodl-tmp/biomni_v20260208_tmp/doubao-seed-1-8/gpedit_refinement"
os.makedirs(intermediate_dir, exist_ok=True)

# Load data
gwas_snps = pd.read_csv("benchmark/dataset/GPEdit/gwas_snps.tsv", sep="\t")
edqtls = pd.read_csv("benchmark/dataset/GPEdit/edqtls.tsv", sep="\t")

# Save intermediate files
gwas_snps.to_csv(os.path.join(intermediate_dir, "gwas_snps_loaded.tsv"), sep="\t", index=False)
edqtls.to_csv(os.path.join(intermediate_dir, "edqtls_loaded.tsv"), sep="\t", index=False)

print("Loaded GWAS SNPs shape:", gwas_snps.shape)
print("Loaded edQTLs shape:", edqtls.shape)

# Perform overlap analysis
# Merge on chromosome first
merged = pd.merge(gwas_snps, edqtls, on="chr", suffixes=("_gwas", "_edqtl"))

# Filter for overlapping positions
overlapping = merged[(merged["pos"] >= merged["start"]) & (merged["pos"] <= merged["end"])]

# Keep only the edQTL columns (as per output example)
result = overlapping[["chr", "start", "end", "edqtl_id", "gene"]].drop_duplicates()

# Save intermediate merged and filtered files
merged.to_csv(os.path.join(intermediate_dir, "merged_data.tsv"), sep="\t", index=False)
overlapping.to_csv(os.path.join(intermediate_dir, "overlapping_all_columns.tsv"), sep="\t", index=False)
result.to_csv(os.path.join(intermediate_dir, "filtered_edqtls.tsv"), sep="\t", index=False)

print("\nOverlapping edQTLs found:", result.shape[0])
print("\nSample overlapping edQTLs:")
print(result.head())

# Save final output
output_path = "/root/autodl-tmp/biomni_v20260208_pred_results/doubao-seed-1-8/gpedit_refinement.tsv"
os.makedirs(os.path.dirname(output_path), exist_ok=True)
result.to_csv(output_path, sep="\t", index=False)

print(f"\nFinal result saved to: {output_path}")
</execute>
